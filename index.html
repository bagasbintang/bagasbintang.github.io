Things I Build With

Tailwind

I use Tailwind because it helps me focus on layout and spacing without overthinking design details. Styling feels faster and more consistent, especially when I want to experiment, tweak things, and see results quickly.

JavaScript

JavaScript allows me to add small interactions and logic to my pages, such as handling user actions or updating content dynamically. it’s what makes a website feel alive instead of just a static page.

Firebase

I use Firebase when I need simple backend features without setting up a full server. It’s helpful for things like basic authentication or storing data, so I can spend more time building rather than dealing with configuration.

React

I’m currently exploring React to better understand how modern user interfaces are structured. I like how component-based thinking helps keep things organized and easier to manage as a project grows, even though I’m still getting comfortable with it.
